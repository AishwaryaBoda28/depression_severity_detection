# 🧠 Depression Severity Detection

A Deep Learning-based web application that predicts the severity of depression from user-input text using Natural Language Processing (NLP) techniques.

---

## 🚀 Features

* 🔍 Predicts depression severity from text input
* 🤖 BERT-based deep learning model
* 🌐 Multilingual input support (auto-translation)
* 📊 Explainability using LIME
* 🖥️ Interactive web interface using Flask

---

## 🛠️ Tech Stack

* **Programming Language:** Python
* **Framework:** Flask
* **Deep Learning:** PyTorch
* **NLP Model:** BERT (Transformers)
* **Explainability:** LIME
* **Libraries:** NumPy, Pandas, Matplotlib

## ▶️ How to Run

### 1. Clone the repository

```
git clone https://github.com/AishwaryaBoda28/depression_severity_detection.git
cd depression_severity_detection
```

### 2. Install dependencies

```
pip install -r requirements.txt
```

### 3. Run the application

```
python app.py
```

---

## 📌 Model Information

* The model is trained using NLP techniques for text classification
* Large model files are not included due to size limitations
* You can retrain the model using:

```
python train.py
```
